## Purpose

SkillSwap is a platform to connect learners with skill providers online.

## Live URL

[Live Demo](https://skillswap-6d6d1.web.app)

## Key Features

- User authentication (Email/Password & Google)
- Skill listing and booking
- User profile management
- Responsive UI

## Npm packages used

- react-router-dom: For routing
- firebase: For authentication and database
- react-hot-toast: For notifications
- react-hot-toast
- animate.css
- tailwindcss/vite
- swiper
- tailwindcss
